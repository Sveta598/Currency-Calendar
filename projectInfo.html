О проекте «Валютный календарь»

Этот проект называется «Валютный календарь». Проект создавался с целью обеспечить пользователю возможность найти интересующую его информацию о курсах иностранных валют по отношению к белорусскому рублю. Для создания этого проекта использовались HTML, CSS и Javascript, в частности html–формы, вёрстка по БЭМ,
